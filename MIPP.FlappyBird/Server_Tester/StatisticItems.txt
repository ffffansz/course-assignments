所有的耗时总和只统计了成功情况下的耗时

连接
    连接成功的机器人数量
    连接成功率 = 连接成功次数 / 总连接次数
    最大连接耗时
    最小连接耗时
    平均连接耗时 = 连接耗时总和 / 连接成功次数

接收SID
    接收SID成功的机器人数量
    接收SID成功率 = 接收SID成功次数 / 总接收次数
    最大接收耗时
    最小接收耗时
    平均接收耗时 = 接收耗时总和 / 接收SID成功次数

注册/登陆
    注册成功的机器人数量
    注册成功率 = 注册成功次数 / 总注册次数
    最大注册耗时
    最小注册耗时
    平均注册耗时 = 注册耗时总和 / 注册成功次数

Notice/RequestRank
    Notice成功率 = Notice成功次数 / 总Notice次数
    最大Notice耗时
    最小Notice耗时
    平均Notice耗时 = Notice耗时总和 / Notice成功次数

完成
    机器人任务完成率 = 执行完所有任务并正常退出的机器人数量 / 机器人总数量
